HRDC Intranet User Documentation

Overview

The HRDC Staff Intranet is an internal platform designed to help HRDC administrators, employees, and volunteers access forms, announcements, and important resources efficiently.

This system is built to be user-friendly, ensuring smooth navigation for all users, including those with limited technical experience.

How to Access the HRDC Intranet

  1. Open your web browser (Chrome, Firefox, Edge, or Safari).
  2. Visit the intranet: HRDC Intranet Website.
  3. Log in if required using your HRDC credentials.
  4. Use the navigation menu or search bar to find the necessary resources.

Using the HRDC Intranet

  • Navigate through the menu to access different sections like "Announcements," "Forms," and "Important Links."
  • Use the search bar at the top of the page to quickly find specific documents or resources.
  • Click on "Help" in the navigation bar to access FAQs and troubleshooting tips.
  • Check the "Announcements" section regularly for updates from HRDC administrators.
  • Log out after using the intranet, especially on shared devices, to ensure your account's security.

Finding Forms & Documents

  • Use the search bar at the top to locate HRDC forms and documents.
  • Click on "Important Links" to quickly access frequently used resources.
  • Files are categorized for easy navigation.

Viewing Announcements

  • Announcements are displayed on the home page and updated by HRDC administrators.
  • HRDC administrators can now edit announcements. New in this release
  • Each announcement now includes the date it was posted, making it easier to track when updates occurred. New in this release

User Roles & Permissions

  • Employees & Volunteers: Can search for documents and view announcements. Can log in with their Gmail account.
  • Administrators: After logging in, Administrators have the ability to create new, edit, or delete announcements and links. To do this, navigate to the Manage Announcements or Manage Links page. On these pages, admins can add, edit, or delete both announcements and links by filling out the forms. (Login: admin@hrdc.com Password: admin123). Updated in this release

Features

  • Home Page: Includes a search bar, latest announcements, quick page links, and calendar of HRDC events.
  • Staff Page: Contains a search bar, staff actions (e.g. timecard submission), HR resources (like the employee handbook), and help info.
  • Help Page: Includes FAQs with dropdowns, a search bar, and a “Need More Help?” section with support contacts.
  • Links Page: Displays a search bar and a list of key HRDC links, now with improved link styling for clarity.
  • Manage Announcements: Admin-only page to post, edit, and delete announcements, and view existing ones. Edit functionality is new in this release
  • Manage Links: Admin-only page to post, edit, and delete links. New in this release
  • Login/Logout: Admins can sign in using email/password or Google login. Logged-in admins see additional features like “Manage Announcements.”
  • Navigation: The top navbar is consistent across all pages and includes links to Home, Staff, Help, and Links. Admins also see a Manage Announcements tab.
  • Search Functionality: Fully functional search bar on most pages, including the Help and Home pages.
  • All internal pages are fully connected for seamless navigation.

Troubleshooting Common Issues

Cannot Access the Intranet

Ensure you are using the correct URL: HRDC Intranet.

If the page does not load, try refreshing or clearing your browser cache.

Always make sure you have a strong enough internet connection.

Forms or Links Are Not Working

Check if the document is listed under "Important Links." If a link is broken, report it using the instructions below.

Ensure you are logged in if the document requires authentication.

If the document is not loading, try refreshing the page or clearing your browser cache.

Login Credentials Not Working

Check capitalization, logins are cap sensitive

Ensure you are using your HRDC-provided login details. If you forgot your password, contact your HR supervisor.

Reporting Issues

  1. Check the Help Page FAQ section for common solutions.
  2. If the issue persists, report it to your supervisor via email or in person.
  3. Include the following details in your report:
    • What action you were trying to perform.
    • The issue encountered (error message, missing content, etc.).
    • The time and date of occurrence.
  4. The HRDC IT team will investigate and resolve the issue.

For technical issues, submit a bug report on GitHub Issues.
