Welcome to the HRDC Intranet

This website is designed to help HRDC employees and volunteers access important resources.

Group 5 Product Website

Our product is an intranet site to assist HRDC employees/volunteers in accessing timecards, forms, and other essential information.

Key Features

  • Search function to browse HRDC Google Drive for forms.
  • Admin functionality to post announcements.
  • Login system for employees/volunteers to access personal documents (W-2s, training files, etc.).
  • Designed for easy use by elderly, visually impaired, and non-native speakers.

User Documentation

The intranet will be connected to HRDC's main website through a subdomain. Users can search for frequently used forms or view announcements.

Developer Documentation

  • Source Code: Available on GitHub.
  • Built using Next.js.
  • Bug tracking is done in GitHub Issues.
  • Code is documented for easy maintenance.

How to Get Help

For questions, contact HRDC support or report issues through GitHub.
